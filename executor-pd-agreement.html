Соглашение о сборе, обработке и передаче персональных данных — для Исполнителя

1. Общие положения

1.1. Оператор — [Данные]. Исполнитель — физическое лицо, регистрирующееся в боте.

1.2. Предмет: обработка персональных данных исполнителя (ФИО, телефон, город, фото/документы при необходимости, статистика заказов, реквизиты для выплат — если согласовано).

2. Цели обработки

2.1. Предоставление доступа к заказам, уведомления, аналитика и статистика, контроль соблюдения правил платформы, связь по вопросам выполнения заказов, выставление/контроль подписки.

3. Передача третьим лицам

3.1. Для исполнения функции платформы оператор может передавать данные другим исполнителям/мастер-админам (только при прямой необходимости), платежным агентам (при оплате подписки) и уполномоченным органам по требованию закона.

3.2. Любая передача происходит с соблюдением принципов минимизации данных и с уведомлением Исполнителя.

4. Права и обязанности Исполнителя

4.1. Исполнитель обязан предоставлять достоверные данные и обновлять их при изменениях; при предоставлении недостоверных данных платформа может ограничить доступ.

4.2. Исполнитель согласен, что его публичная контактная информация (имя/телефон/город) может быть видна Заказчику после принятия заказа.

5. Согласие

Регистрируясь и оплачивая подписку, я даю согласие на обработку моих персональных данных... в целях работы в рамках платформы, включая передачу данных Заказчику при принятии заказа и платежному агенту при оплате подписки.
